Um simples jogo de advinhação escrito em python feito para uma atividade
da matéria de Estrutura de Dados Avançada do curso de Ciência da Computação
da UNIMA

# Estrutura da árvore de decisão

```
                 O seu pokémon é 
                 do tipo elétrico?
                 /           \
              Sim             Não
              /                \
       O seu pokémon é      O seu pokémon   
       da região de Kanto?  é um lendário?               
        /     \              /     \
    Pikachu  Rotom       Mewtwo  Charizard

```
# Como rodar o código
* Primeiramente, certifique-se de que você tem o python 3.12 rodando
  o seguinte comando no terminal:
  
  `python3 --version`

* Após isso é só rodar esse comando na pasta onde você baixou o repositório:

  `python3 JogoDeAdvinhacao.py`
